# apimockserve

## Descripition

supply api serve and mock serve

## Get Start

```
git clone git@mzh-gitlab.com:shenpy/apimockserve.git

cd apimockserve

npm install or yarn

# convert raml to html
npm run raml


npm run serve
npm run mock

or

npm run mock-serve

```

## Serve

- api serve on http://localhost:8080.

* mock serve on http://localhost:8081.
